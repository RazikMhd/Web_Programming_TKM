GIT Functions
1. installed git using package manager.
2. Initialized the workspace using 'git init' command
3. staged the changes using 'git add .' 
4. made a commit using 'git commit -m "commit message"'
5. push the commited files to github repository using 'git push origin master'
6. when prompted for credentials, provide the same.

GIT Key-Pairing
1. generate a key using 'ssh-keygen', and provide a passphrase.
2. go to ssh location in your system, find the newly generated key and copy its contents.
3. Go to gitHub, Settings>SSH & GPG keys>Add Key
4. Copy Paste the key value into the field given.
5. type the 'ssh -T git@github.com' and provide the passphrase.
